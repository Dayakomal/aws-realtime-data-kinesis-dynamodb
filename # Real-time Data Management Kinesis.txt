# Real-time Data Management: Kinesis → Lambda → DynamoDB

**Architecture:** Kinesis Data Stream → AWS Lambda → DynamoDB  
**Region:** ap-south-1 (Mumbai)

## What this repo contains
- `src/producer`: Python script to push records into Kinesis
- `src/lambda`: Lambda function code to read from Kinesis and write to DynamoDB
- `infra/iam`: IAM policies/roles JSON
- `docs/screenshots`: All project screenshots
